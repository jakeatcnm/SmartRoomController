# SmartRoomController

## Overview
The purpose of this project is to adjust the lights in a room.

1. Turn on lights
1. Sense light levels
1. Allow user to manually adjust lights

## Details

This project requires the following components:

* Phillips Hue Lights
* Breadboard
* Wires
* Teensy 3.2 Microcontroller
